# Get started with Multipass 

Simple script to get Multipass Ubuntu-VMs running in a jiffy

## Install

Ref: https://multipass.run/

1. git clone https://github.com/ch3332/MULTIPASS
2. cd MULTIPASS
3. chmod  +x  *.sh
4. ./setup.sh

## Launch VMs

Multipass Custom VM build

1. ./launch_vm.sh
